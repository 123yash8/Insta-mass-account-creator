- Requirements:<br>
  - run `pip install -r requirements.txt`
  - create a new firebase account <br>
  - got to databases <br>
  - click on rules <br>
  
Change Settings to
```
 {
  "rules": {
    ".read": true,
    ".write": true
  }
}
```

Download chrome driver<br> 
configure it to path<br>

update config file to what suites your needs<br>
